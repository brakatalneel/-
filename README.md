تجارة اللحوم التي  ترعي في المراعي الطبيعية ونحن نقوم باستيراد اللحوم وفق. متطلبات الهيئة العامة للغذاء والدواء السعودية استيراد اللحوم السودانية والاثيوبية والحيوية والتي تذبح وفق معايير الجوده العاليه والذبح الحلال نانواع اللحوم هي الضأن والماعز والبقر اللحم المفروم والبترول والفرقان المبرد ومحفوظ بطريقه صحيحه وصحية كما نقدم خدمة العملاء في الحص
